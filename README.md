# Texas Claude Builder Club Official Website

We try and match Anthropic's conversational and friendly design philosophy. We prioritize readability and functionality, making it as easy as possible for 
visitors to join the club, and for members to find what they need.

This site is hosted on Vercel. If analytics are enabled, you can view how many users
visit the site!

## Setup

Navigate to the frontend directory: `cd frontend`

Install dependencies: `npm install`

Create .env file and add the following keys:
- NEXT_PUBLIC_GOOGLE_CALENDAR_API_KEY
- NEXT_PUBLIC_GOOGLE_CALENDAR_ID

See .env.example for reference.

Run the development server: `npm run dev`


## Tech Stack

- Next.js
- Tailwind CSS
